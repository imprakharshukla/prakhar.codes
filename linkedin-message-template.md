# LinkedIn Message Templates

## Template 1: Direct Message

```
Hi {name},

Built Andronix to $140K+ revenue and 2.5M+ downloads, plus Lumoflo (multi-tenant e-commerce SaaS). Most recently founding engineer at Yobr in Norway.

Looking for my next early-stage opportunity - would love to connect and learn more about what you're building.

prakhar.codes

Best,
Prakhar
```

### Variables to Fill
- `{name}` - Recipient's first name

---

## Template 2: Connection Request Note (Same College)

```
Hi {name}, fellow {college_name} alum here! Saw you're at {company_name}. Built Andronix ($140K revenue, 2.5M+ downloads) + Lumoflo, was founding eng at Yobr (Norway). Exploring opportunities - would love to connect!
```

### Variables to Fill
- `{name}` - Recipient's first name
- `{college_name}` - Your shared college name
- `{company_name}` - Company where they work

**Note:** LinkedIn connection requests are limited to 300 characters, so keep it brief!
